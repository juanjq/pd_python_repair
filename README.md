# pdfpy
A script to edit pdfs in different ways. Installed with,
```
import httpimport
with httpimport.remote_repo(['pdfpy'], 'https://raw.githubusercontent.com/juanjq/pdfpy/main'):
     import pdfpy 
```
For this we need installed `PyPDF2`, using `pip install PyPDF2`
